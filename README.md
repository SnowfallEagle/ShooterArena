# Shooter Arena

## Summary
![](Project/Images/Header.png)

Shooter Arena is single-player Quake-like arena game build with Unreal Engine 5.
You can play it here: [Shooter Arena Itch Io](https://seregaengine.itch.io/shooter-arena).
